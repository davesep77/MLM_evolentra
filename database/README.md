# Database Schema Directory

This directory contains database schema files for deployment.

## Files

- `schema.sql` - Main database schema export (generated by export script)
- `backup_*.sql` - Timestamped backup files

## Exporting Database

### Windows (XAMPP)
```bash
export_database.bat
```

### Linux/Mac
```bash
chmod +x export_database.sh
./export_database.sh
```

### Manual Export
```bash
# Using mysqldump
mysqldump -u root -p --databases evolentra > database/schema.sql

# Or from MySQL Workbench
# Server → Data Export → Select evolentra → Export to Self-Contained File
```

## Importing to DigitalOcean

### Method 1: Database Console (Web UI)
1. Go to DigitalOcean database dashboard
2. Click "Console" tab
3. Copy and paste SQL from `schema.sql`
4. Execute

### Method 2: MySQL Client
```bash
# Get connection details from DigitalOcean dashboard
mysql -h your-db-host -P 25060 -u doadmin -p your-database-name < database/schema.sql
```

### Method 3: MySQL Workbench
1. Create new connection with DigitalOcean credentials
2. File → Run SQL Script
3. Select `schema.sql`
4. Execute

## Verification

After import, verify tables exist:

```sql
SHOW TABLES;
SELECT COUNT(*) FROM users;
```

- transactions
- referrals

- support_tickets
